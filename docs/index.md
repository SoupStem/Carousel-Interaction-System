# **Welcome!**

Thank you for purchasing the Carousel Interaction System (CIS) Plugin!

Your support is truly appreciated. CIS was built with care and attention to detail to provide a flexible, optimized, and easy-to-use plugin for creating immersive interaction and inventory system in Unreal Engine.

We hope this plugin helps you bring your projects to life and inspires new creative possibilities.


<u>Features</u>:

- Carousel Inventory Layout – V-shaped visual layout with focus on the central item.

- Advanced Interaction System – Interact easily with world objects, pickups, and UI.

- Scene Capture Rendering – Real-time item visualization in menus or inventory screens.

- Data-Driven Setup – Uses Data Assets, Data Tables, and Gameplay Tags.

- C++ Optimized Core – Written in clean, documented, and efficient C++.

- Blueprint Accessible – Exposed functions for easy customization and prototyping.

- Plug & Play Integration – Simple to add and ready to use in any Unreal project.
